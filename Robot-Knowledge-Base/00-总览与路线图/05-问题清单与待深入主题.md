# 问题清单与待深入主题

## 1. 使用方式

本文是全局问题入口，用来记录学习过程中还没想明白、需要实验验证或后续深入的主题。

建议分工：

- 本文记录“全局问题”和“待深入主题索引”；
- 具体报错放到 [[14-问题库/03-报错与解决记录]]；
- 已解决问题沉淀到 [[14-问题库/02-已解决问题]]；
- 复盘经验沉淀到 [[14-问题库/05-问题复盘与经验]]。

问题记录格式建议：

```text
问题：我不理解什么？
场景：在哪个主题/项目里遇到？
当前理解：我已经知道什么？
下一步：准备查资料、做实验还是问别人？
状态：待解决 / 处理中 / 已解决
```

## 12. 每周复盘时要更新

每周复盘时检查：

- 本周新增了哪些问题；
- 哪些问题已经解决；
- 哪些问题其实是同一个问题的不同表述；
- 哪些问题应该拆成更小的问题；
- 哪些问题暂时超出“6 成了解”的范围，可以延后。

优先处理会阻塞主线学习的问题，暂时不要被高级细节拖住。
