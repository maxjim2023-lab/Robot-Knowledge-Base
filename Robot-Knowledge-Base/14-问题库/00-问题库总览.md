# 问题库总览

## 1. 使用方式

- 新出现的问题先记录到 `01-待解决问题.md`
- 已解决的问题沉淀到 `02-已解决问题.md`
- 命令、环境、依赖、启动类报错记录到 `03-报错与解决记录.md`
- 暂时不影响当前进度、但值得深入的主题记录到 `04-待深入主题.md`
- 周复盘时把高价值问题整理到 `05-问题复盘与经验.md`

## 2. 问题分类

| 分类 | 说明 | 推荐记录位置 |
|---|---|---|
| 概念不清 | 原理、术语、系统链路没有理解透 | `04-待深入主题.md` |
| 实验失败 | 仿真、编译、启动、运行结果异常 | `01-待解决问题.md` |
| 报错排查 | 有明确错误日志或命令输出 | `03-报错与解决记录.md` |
| 已解决问题 | 已经有稳定解决方案 | `02-已解决问题.md` |
| 经验沉淀 | 可复用的判断、排查路径、避坑点 | `05-问题复盘与经验.md` |
